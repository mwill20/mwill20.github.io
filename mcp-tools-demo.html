MCP Tools Demo

GitHub Repository Live Demo on Hugging Face

Technology Stack

Python Gradio 5.32.1 smolagents 1.17.0 Model Context Protocol Hugging Face Spaces TextBlob

Project Overview

The MCP Tools Demo is a multi-tool Gradio application that demonstrates the integration of Model Context Protocol (MCP) tools. This application showcases how AI tools can be exposed through a standardized protocol, making them accessible both through a user interface and programmatically via API endpoints.

Features

  • Sentiment Analysis Tool: Analyzes the sentiment of text and provides polarity and subjectivity scores using TextBlob.
  • Weather Tool: Provides weather information for a specified location (mock implementation for demonstration).
  • Calculator Tool: Performs basic arithmetic operations (add, subtract, multiply, divide).
  • MCP Integration: All tools are exposed via the Model Context Protocol using smolagents.
  • Gradio UI: User-friendly interface for interacting with the tools.

Technical Implementation

The application uses the @tool decorator from smolagents.tools to expose Python functions as MCP tools. These tools are then integrated into a Gradio interface, which provides both a web UI and API endpoints.

The application is configured to run both locally and in a Hugging Face Spaces environment, with environment variables controlling the server configuration and MCP enablement.

Deployment

The application is deployed on Hugging Face Spaces, making it accessible to anyone with an internet connection. The deployment process involved pushing the code to GitHub and linking the repository to Hugging Face Spaces.

Challenges and Solutions

One of the main challenges was ensuring compatibility between Gradio 5.32.1 and smolagents 1.17.0. This required careful implementation of the @tool decorator and proper error handling for MCP initialization.
