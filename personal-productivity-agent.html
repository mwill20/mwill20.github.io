Personal Productivity Agent

2024
Python MCP Windows Automation Gemini LLM

Overview

I'm excited to share a project I've been developing: a Personal Productivity Agent for Windows! This Python-based tool is designed to help automate various tasks, provide system insights, and leverage the power of Google's Gemini Large Language Model (LLM) for intelligent analysis.

Key Features

Smart Event Log Analysis

Fetch and analyze Windows Event Logs using Gemini LLM to provide explanations and fixes.

File Management

Easily find, move, rename files, and create directories with interactive confirmations.

System Insights

Get detailed information about disk usage, memory, CPU, and network status.

AI-Powered

Leverages Google's Gemini LLM for intelligent task automation and analysis.

Two Modes of Operation

Interactive CLI

Run python main_agent.py for a user-friendly menu interface to access all features directly from your terminal.

MCP Server Mode

Run python main_agent.py --mcp-mode to transform the agent into a backend server that can be controlled programmatically by other applications.

MCP Integration

In MCP (Multi-Capability Protocol) Server Mode, the agent can:

  • Process commands in a standardized JSON format
  • Integrate with other MCP-compatible agents and services
  • Connect to the MCP Core Server & AI Model Hub for enhanced capabilities
  • Enable complex, multi-agent workflows and automation
